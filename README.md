# TWQR 收款碼產生器 (TWQR Generator)

一個使用 Next.js、React 和 Tailwind CSS 開發，免費且開源的台灣 Pay (TWQR) 收款 QR Code 產生器！

## ✨ 主要功能

* **支援多數金融機構**：內建完整的銀行代碼列表（包括農漁會、行動支付錢包等）
* **自訂收款資訊**：輕鬆設定收款銀行、帳號、金額與備註
* **客製化 Logo**：支援在 QR Code 中央嵌入 TWQR 圖示或您自行上傳的 Logo
* **安全性**：所有的 QR Code 都在您的瀏覽器中即時產生，伺服器不會儲存任何您輸入的資料

## 🚀 如何在本機運行

1. **下載/複製專案**
   ```bash
   git clone [https://github.com/HappyGroupHub/twqr-generator.git](https://github.com/HappyGroupHub/twqr-generator.git)
   cd twqr-generator
   ```

2. **安裝依賴套件**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **啟動開發伺服器**
   ```bash
   next dev
   ```

4. 完成！打開瀏覽器並前往 `http://localhost:3000`
